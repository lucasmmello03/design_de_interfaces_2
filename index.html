O que são Funções?

Trata-se da relação entre um conjunto não vazio e outro conjunto também não vazio, na qual cada elemento do primeiro grupo relaciona-se com um único elemento do segundo.

É uma aplicação relacionando os elementos presentes em dois conjuntos não vazios. Considere os conjuntos A e B, onde a regra matemática relaciona cada item pertencente a A a um único correspondente em B.

Conceitos Básicos

  • Domínio: Grupo com todos os valores aplicados na entrada (x).
  • Imagem: Subconjunto do contradomínio formado pelos elementos que possuem correspondente.

Esses modelos matemáticos são essenciais para prever comportamentos, como o crescimento populacional ou a trajetória descrita por um objeto.

Classificações Principais

  • Injetora: Cada item no conjunto A possui um correspondente exclusivo no conjunto B.
  • Sobrejetora: O grupo Imagem equivale ao Contradomínio (não sobra ninguém em B).
  • Bijetora: Quando a regra atende aos dois casos anteriores simultaneamente.

Primeiro Grau

Definida pela lei f(x) = ax + b. Seu gráfico será sempre uma reta. Quando a > 0 a reta sobe, quando a < 0 a reta desce.

Exemplo: f(x) = 2x + 3. Para x = 1, o resultado é 5.

Segundo Grau

Definida pela lei f(x) = ax² + bx + c. Seu gráfico forma uma parábola. Quando a > 0 a parábola abre para cima, quando a < 0 abre para baixo.

Exemplo: f(x) = x² − 4. As raízes são x = 2 e x = −2.

Exponencial

Determinada pela lei f(x) = aˣ. Aparece em situações envolvendo crescimento rápido, como juros compostos e expansão populacional.

Exemplo: f(x) = 2ˣ. Para x = 3, o resultado é 8.

Logarítmica

Trata-se do modelo inverso ao exponencial. Usada em marcações como a escala pH e a escala Richter.

Exemplo: log₁₀(1000) = 3, porque 10³ = 1000.
